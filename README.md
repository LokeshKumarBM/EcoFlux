# ⚡ EcoFlux — Smart Microgrid Renewable Energy Monitoring System

EcoFlux is an IoT-based renewable energy monitoring and management system designed for microgrids. It provides real-time insights into power generation, storage, and consumption using an integrated mobile platform, enabling efficient control and improved reliability.

---

## 🚀 Features

* 🔌 Real-time monitoring of solar, battery, and load data  
* ⚡ Smart load prioritization for uninterrupted power  
* 🔄 Hybrid system support (solar + battery + diesel)  
* 📡 IoT-based data collection and remote control  
* 🚨 Instant fault detection and alerts  
* 📱 Mobile app interface with multi-language support  
* 🔁 Offline + online data synchronization  

---

## 🛠️ System Components

* ESP32 Microcontroller  
* Sensors for energy and system monitoring  
* Battery Management System (BMS)  
* Mobile application interface  

---

## 💡 Key Innovation

EcoFlux integrates multiple functionalities into a single platform:

* Real-time monitoring  
* Smart load management  
* Direct bypass operation  
* Immediate fault alerts  

This provides comprehensive control over microgrid systems, unlike traditional solutions.
